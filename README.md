뉴스 요약이 아닙니다. DART(전자공시)·KRX(공매도)·Naver(시세·외국인)·국민연금을 **직접 호출**해, 흩어진 데이터를 하나의 투자 판단으로 엮어 **인용과 함께** 답합니다. 미국 종목도 같은 자리에서 묻고 비교할 수 있습니다.

> **"삼성전자 지금 투자 관점에서 어때?"** -> 이 한 줄이면 실적·외국인 수급·공매도 잔고·대량보유·목표주가 컨센서스를 각각 출처에서 끌어와, 신호들이 같은 방향인지 충돌인지까지 정리해 줍니다.

## 뭐가 다른가

DART·KRX를 보여주는 도구는 이미 많습니다. 차이는 **그 위에서 도는 에이전트 루프**에 있습니다.

- **스스로 찾아옵니다.** 질문을 받으면 어떤 1차 출처가 필요한지 에이전트가 판단해 도구를 골라 호출합니다. 사용자가 DART·KRX를 일일이 뒤질 필요가 없습니다.
- **물어본 것 이상을 봅니다.** "삼성전자 어때?" 한 줄이면 재무·수급·공매도·대량보유·공시에 더해, 묻지 않은 **동종 peer 비교**와 **사업부문별 이익 기여도**까지 스스로 끌어와 맥락을 만듭니다.
- **엮어서 답합니다.** 지표를 나열하는 게 아니라 실적·수급·지배구조·공시를 하나의 thesis로 종합하고, 근거에 기반한 결론과 트리거로 닫습니다.
- **인용하고 재현됩니다.** 모든 호출이 기록되어 "무엇을 어디서 가져왔는지" 추적·감사할 수 있습니다.
- **조용히 안 깨집니다.** 한도 초과나 응답 변경으로 값이 비면, 틀린 숫자를 답하는 대신 경고를 붙입니다.

한마디로, Frontier LLM의 능력치를 **한국 금융 하네스**로 최대한 끌어내는 제품입니다. 그 위에 DCF·상대가치·지주사 SOTP·주주환원·물적분할 같은 **전문가 분석 스킬**이 질문에 따라 자동으로 얹힙니다.

## 이런 걸 물어보세요

시장을 지정할 필요 없이 한국어로 그냥 질문하시면 됩니다. (6자리 숫자 티커는 한국으로 자동 인식)

**한 줄 종합 — 루프가 가장 빛나는 질문**

```
삼성전자 지금 투자 관점에서 어때?
↳ 실적·외국인 수급·공매도·대량보유·목표주가를 한 번에 끌어와 교차 종합
```

**비교 · 랭킹 — 루프를 끝까지 미는 질문**

```
삼성전자·SK하이닉스·한미반도체 밸류에이션·수급·실적 비교해서 투자매력 순위 매겨줘
↳ 종목마다 재무·세그먼트·현금흐름·수급·공매도·지배구조를 각각 수집해 6개 차원 표 + 랭킹
```

**밸류에이션**

```
SK하이닉스 DCF로 적정주가 계산하고 현재가랑 비교해줘
SK 지주사 SOTP로 적정가치 뜯어줘                   ← 지주사 할인 분해
삼성전자 주주환원(배당·자사주) 정책이 주가에 주는 의미 분석해줘
엔비디아랑 SK하이닉스 HBM 실적 비교해줘            ← 한·미 통합
```

**공시 · 지배구조 (DART 본문에서 직접)**

```
카카오 최근 1년 주요 공시 정리해줘
삼성전자 최대주주·특수관계인·계열회사 관계 사업보고서 기준으로 정리해줘
LG화학 물적분할이 기존 주주가치에 어떤 영향이었는지 분석해줘
```

**한국 특화 데이터**

```
삼성전자 현재가랑 목표주가 컨센서스 알려줘          ← 키 없이 동작
삼성전자 외국인 지분율 최근 추세 보여줘            ← 키 없이 동작
에코프로비엠 공매도 순보유잔고 추이 어때?
국민연금이 보유한 삼성전자 지분 알려줘
```

**메모로 정리**

```
삼성전자 매수 논거를 투자 메모로 작성해줘
```

## 웹 챗봇이랑 뭐가 달라?

같은 질문을 일반 AI 챗봇(+웹검색)과 비교해 보면, 차이는 모델이 아니라 **1차 출처를 직접 호출하는 하네스**에서 옵니다.

| | 일반 챗봇 (+웹검색) | Dexter |
|---|---|---|
| 출처 | 뉴스·블로그 2차 | **DART·KRX·Naver 1차** |
| 공매도 · 외국인 | 1개 시점 + "직접 조회하세요" | **수십 일 일별 시계열** |
| 밸류에이션 | 일반 가정 | **한국 시장 자동 분기** (법인세·무위험금리·KRW) |
| 신뢰 | 가끔 환각 | **인용 + 재현 가능** |

## 시작하기

키가 하나도 없어도 **한국 종목 현재가·외국인 지분율(Naver)** 은 바로 동작합니다. 재무·공시·공매도·연기금 같은 깊은 리서치는 해당 키를 넣으면 자동으로 켜집니다.

- **데스크톱 앱 (Windows/macOS)** — 맨 위 **⬇ 다운로드** 버튼으로 받아 설치한 뒤, 설정 화면에서 키만 입력하면 끝.
- **CLI · 개발자** — Bun으로 바로 실행. 설치·키·도구 상세·견고성·평가까지 전부 → **[개발 문서](DEVELOPMENT.md)**

## 정직한 한계

항상 이기는 도구는 아닙니다.

- 은행·지주사처럼 회계 라벨이 비표준인 곳은 일부 지표가 비어 raw 드릴다운으로 보완 — 커버리지가 균일하지 않습니다.
- 브로커별 목표주가 분산·세그먼트 재무 같은 정밀 데이터는 유료 단말(FnGuide 등)에 못 미칩니다.
- KRX는 로그인, Naver는 비공식 엔드포인트에 의존 — 견고성 레이어가 충격은 줄이지만 상류 변경 자체를 막진 못합니다.

우위는 "더 똑똑함"이 아니라 **1차 출처 · 구조화 · 감사 가능 · 한국 튜닝**에 있습니다.
