# 한국 주주환원 / 밸류업 분석 스킬

목표: 대상이 **밸류업(기업가치제고) re-rating 후보**인지, 즉 주주환원 강화로 코리아 디스카운트(저PBR)가 좁혀질 여지가 있는지 판단한다. 결론(re-rating 후보 여부 + 핵심 근거)을 먼저. **PBR<1을 단순 "통계적 저평가"로 끝내지 마라 — 환원 의지/실행이 있어야 디스카운트가 풀린다. 환원 약속의 부재 자체가 디스카운트 지속의 근거다.**

## 배경 (2024~)

정부 '기업 밸류업 프로그램'으로 저PBR·고잉여현금 기업의 **기업가치제고계획** 자율공시, 자사주 **소각**(매입만으로는 부족 — 소각해야 주식수 감소), 배당 확대가 핵심 테마가 됐다. 자사주는 **매입 후 소각**해야 EPS·주주가치가 실제로 오른다(매입 후 보유는 추후 재매각 리스크).

## 워크플로 체크리스트

```
KR 주주환원/밸류업 분석 진행:
- [ ] Step 1: 밸류에이션·환원 현황 스냅샷
- [ ] Step 2: 기업가치제고계획·자사주·배당정책 공시 탐색
- [ ] Step 3: 환원 여력(현금·별도이익) 점검
- [ ] Step 4: re-rating 판단 + 트리거
```

## Step 1: 스냅샷

대상 티커(또는 회사명)로 `get_market_data_kr` → `valuation.{pbr, dividendYieldPct, dividendPerShare, marketCap}`. **PBR<1이면 밸류업 1차 후보.** 배당수익률 수준과 52주 위치도 기록. PBR이 1을 크게 넘으면 밸류업 앵글은 약하므로 일반 밸류에이션으로 전환.

## Step 2: 환원 공시 탐색 (핵심)

`get_filings_kr`로 환원 관련 공시를 찾는다(기본 범위 ~1년, 필요시 `start_date` 확대):
- `filing_type: "exchange"`(거래소공시) → **기업가치제고계획** 자율공시.
- `filing_type: "material"`(주요사항보고) → **자기주식 취득/소각**, 자기주식취득 신탁계약.
`report_nm`에서 `기업가치제고`, `자기주식`, `자사주`, `소각`, `배당` 키워드 스캔. 배당정책 본문은 `read_filings_kr`(sections `["overview"]` 또는 `["governance"]`)로 보완. 공시가 부족하면 `web_search`로 최근 IR/밸류업 발표 확인.

## Step 3: 환원 여력

`get_financials_kr`로 환원을 감당할 체력을 본다:
- **순현금**(`balanceSheet.cashAndEquivalents + shortTermInvestments − totalDebt`)이 두둑하면 자사주 소각·특별배당 여력.
- **FCF**(`ratios.freeCashFlow`)가 흑자 지속이면 지속가능 환원.
- **배당성향** = 배당총액 ÷ `incomeStatement.netIncome`(또는 DPS×주식수 ÷ NI). 배당재원은 **별도(OFS)** 이익이므로 별도 이익도 확인.
낮은 배당성향 + 두둑한 순현금 = 환원 여력은 큰데 안 하는 상태 → 밸류업 압박의 표적이자 잠재 upside.

## Step 4: re-rating 판단 + 출력

1. **평가**(1~2문장): 밸류업 re-rating 후보 여부 + 가장 큰 근거(공시 실행 / 여력 / 부재).
2. **현황**: PBR, 배당수익률·성향, 순현금/FCF.
3. **환원 트랙레코드**: 기업가치제고계획·자사주 소각·배당 공시 유무와 내용(매입 vs **소각** 구분).
4. **판단**: 환원 의지+여력+실행이면 디스카운트 축소 트리거; 여력은 있으나 무행동이면 디스카운트 지속 근거로 명시.
5. **트리거/주의**: "자사주 소각 공시", "기업가치제고계획 상향", "배당성향 목표 제시" 등 구체 트리거. 코리아 디스카운트는 일부 구조적(지배구조·세제)이라 환원만으로 완전 해소되진 않음. 배당세(거주자 15.4%, 외국인 ~22% 또는 조세조약) 세후 별도. (지주사면 `kr-sotp-holdco`와 함께 보라.)
